Sistem Pendaftaran Acara ini merupakan aplikasi berbasis web yang dikembangkan menggunakan Laravel 12. 
Aplikasi ini dirancang untuk memfasilitasi proses pendaftaran peserta acara secara digital, serta 
memudahkan admin dalam mengelola data peserta secara terstruktur.

Sistem ini memiliki dua peran utama, yaitu Peserta (User) dan Admin. 
Peserta dapat melakukan pendaftaran acara melalui form yang telah disediakan dengan mengisi data lengkap, 
sedangkan Admin dapat mengelola data pendaftaran peserta melalui dashboard admin yang dilengkapi 
dengan fitur CRUD dan filter acara berdasarkan bulan pelaksanaan.

Antarmuka aplikasi dirancang menggunakan konsep Dark Modern UI dengan pendekatan glassmorphism 
untuk memberikan pengalaman pengguna yang nyaman, modern, dan profesional. 
Sistem ini cocok digunakan sebagai studi kasus pengembangan web, tugas perkuliahan, maupun portofolio.
